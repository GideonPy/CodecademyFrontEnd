Colors

Color Schemes:

Different arrangements of colors on the color wheel give you different feelings or moods:

  • Monochromatic: Uses variations in lightness and saturation of a single color.
  • Analogous: Uses colors that are nect to each other on the color wheel.
  • Conplementary: Uses colors opposite each other on the color wheel.
  • Triadic: Uses three colors evenly spaced around the color wheel.
  • Tetradic(Double Complementary): Uses two pairs of complementary colors

Hexidecimal Color Codes:

In web design, colors are often represented using hexadecimal(hex) codes, which are 6-characters strings preceded by a hash(#). For Example:

  • #FFFFFF is white
  • #000000 is black
  • #FF0000 is red

RGB&RGBA:

Colors can also be represented using the RGB(Red,Green,Blue) model:

  • 'rgb' (255,0,0) is red.
  • 'rgba' is similar but includes an alpha(a) valuw to represent opacity. For example:
  • 'rgba' (255,0,0,0.5) is half-transparent red.

Color Psycology:

Different colors can evoke different feelings and perceptions:

  • Red: Passion, excitment,danger.
  • Blue: Trustworthiness, calm, sadness.
  • Yellow: Happiness,optimism,caution.
  • Green: Natural, growth, envy.

Contrast:

It's crucial for readability. There should be a distinct contrast between text color and its background. This is especially vital for accessibility reasons, ensuring that those with visual impairments can still interact with web content.

Consistency:

Maintaining a consistent color palette across a website helps in brand recognition and user expwerience. Typically,designers choose primary, secondary and accent colors to maintain this consistency.

Tools & Resources:

There are many online tools to help with color choices:

  • Adobe color: Helps in creating color schemes.
  • Coolors.co: Generates color paletts.
  • ColorZilla: A browser extension to pick colors from web pages.
  • HTML Color Picker: This is a webpage that has all the availible colors in HTML5. It will give you the Hexidecimal, HSL, RGB, and RGBA of that color.
